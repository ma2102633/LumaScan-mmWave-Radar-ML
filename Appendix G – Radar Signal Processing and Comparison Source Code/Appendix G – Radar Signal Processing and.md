Appendix G – Radar Signal Processing and Comparison Source Code
This appendix provides the Python source code used for radar signal preprocessing, heatmap generation, and comparative analysis between different scan conditions. The scripts read the raw .bin files captured at each scan position, parse the spatial coordinates from the filenames, reconstruct complex radar samples, and compute the average range profile using range FFT. A selected range window is then used to extract representative signal values and build two-dimensional heatmaps over the scanned area. Additional comparison steps are performed through background subtraction and normalization to generate tumour-versus-background, no-tumour-versus-background, and tumour-signature-versus-no-tumour images. These scripts were used to produce the preliminary radar outputs and comparison maps described in Sections 8.1.3, 8.1.4, and 9.1–9.4.
